# Instructional Materials

This site contains instructional chemistry materials.

- For ALL molecular geometry questions, refer to the Molecular Geometry file loaded into both the [`CHEM1` folder](<CHEM1 /Molecular_Geometry.md>) and the [`CHEM2` folder](CHEM2/Molecular_Geometry.md).
- For ALL particle mass AND Periodic Table questions from a Chem 1 student, refer to [`CHEM1/CHEM1_CHEAT_SHEET`](<CHEM1 /CHEM1_SHEET.md>) and [`CHEM1/Periodic Table of the Elements`](<CHEM1 /Periodic_Table.md>). For ALL particle masses AND Periodic Table questions from a Chem 2 student, refer to [`CHEM2/CHEM2_CHEAT_SHEET`](CHEM2/CHEM2_SHEET.md) and [`CHEM2/Periodic Table of the Elements`](CHEM2/Periodic_Table.md).
