# Equations

\[
\nu = \frac{c}{\lambda}
\]

\[
E = h\nu
\]

\[
\lambda = \frac{h}{mv}
\]

\[
\Delta x(m\Delta v) \geq \frac{h}{4\pi}
\]

\[
E = \frac{q_1q_2}{4\pi\varepsilon_0r}
\]

\[
\Delta E = -2.18 \times 10^{-18}\ \text{J}
\left[
\frac{1}{n_f^2} - \frac{1}{n_i^2}
\right]
\]

# Useful Conversion Factors and Relationships

1 lb = 453.6 g  
1 in = 2.54 cm (exactly)  
1 mi = 1.609 km  
1 km = 0.6215 mi  
1 pm = $1 \times 10^{-12}$ m = $1 \times 10^{-10}$ cm  
1 atm = 760 mmHg = 760 torr = 101,325 N/m$^2$ = 101,325 Pa  
1 cal = 4.184 J (exactly)  
1 L · atm = 101.325 J  
1 J = 1 C × 1 V  

\[
^\circ C = (^\circ F - 32^\circ F) \times \frac{5^\circ C}{9^\circ F}
\]

\[
^\circ F = \frac{9^\circ F}{5^\circ C} \times (^\circ C) + 32^\circ F
\]

\[
K = (^\circ C + 273.15^\circ C) \left(\frac{1 K}{1^\circ C}\right)
\]

# Solubility Rules

## Compounds Containing These Ions Are Generally Soluble

| Ions                                | Exceptions                                  |
|------------------------------------|---------------------------------------------|
| $\text{Li}^+, \text{Na}^+, \text{K}^+, \text{NH}_4^+$ | none                                        |
| $\text{NO}_3^-, \text{C}_2\text{H}_3\text{O}_2^-$     | none                                        |
| $\text{Cl}^-, \text{Br}^-, \text{I}^-$                | $\text{Ag}^+, \text{Hg}_2^{2+}, \text{Pb}^{2+}$ |
| $\text{SO}_4^{2-}$                                     | $\text{Ag}^+, \text{Ca}^{2+}, \text{Sr}^{2+}, \text{Ba}^{2+}, \text{Pb}^{2+}$ |

## Compounds Containing These Ions Are Generally Insoluble

| Ions                                | Exceptions                                                                 |
|------------------------------------|----------------------------------------------------------------------------|
| $\text{OH}^-$                      | $\text{Li}^+, \text{Na}^+, \text{K}^+, \text{NH}_4^+$; $\text{Ca}^{2+}, \text{Sr}^{2+}, \text{Ba}^{2+}$ are slightly soluble |
| $\text{S}^{2-}$                    | $\text{Li}^+, \text{Na}^+, \text{K}^+, \text{NH}_4^+, \text{Ca}^{2+}, \text{Sr}^{2+}, \text{Ba}^{2+}$ |
| $\text{CO}_3^{2-}, \text{PO}_4^{3-}$ | $\text{Li}^+, \text{Na}^+, \text{K}^+, \text{NH}_4^+$                      |
